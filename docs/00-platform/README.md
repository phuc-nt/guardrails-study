# 00 — Platform

Vision, roadmap dài hạn, principles, multi-tenant model, watchlist giải pháp guardrail của Guardrail Platform toàn tập đoàn.

## Files dự kiến

- `vision.md` — vì sao tập đoàn cần guardrail platform; not just API server
- `roadmap.md` — rolling roadmap, milestones, planned consumers
- `principles.md` — design principles (multi-tenant, plugin-first, fail-open policy, observability)
- `multi-tenant-model.md` — tenant config / policy / scanner pack / audit log layout
- `solution-watchlist.md` — danh mục giải pháp đang theo dõi (NeMo, llm-guard, Guardrails-AI, Bedrock Guardrails, Azure Content Safety, OpenAI Moderation, Anthropic Constitutional AI, …) — review định kỳ
- `glossary.md` — thuật ngữ chung

## Lưu ý

Các file ở đây là **design/strategy documents** dành cho team nội bộ tham chiếu khi triển khai. Không có execution artifact (benchmark, runbook) ở repo này.
